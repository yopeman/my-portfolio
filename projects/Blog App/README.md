The **graph_blog** project is a collection of three GraphQL server implementations that demonstrate the same API with Apollo Server, Ariadne, and Strawberry.

### What it solves
It solves the challenge of comparing GraphQL stack choices by providing a consistent schema and data model across Node.js and Python implementations.

### Technology Stack
* Apollo Server 5, Node.js
* Ariadne, Python 3.12+
* Strawberry GraphQL, Python 3.12+
* Sequelize, SQLAlchemy
* SQLite
