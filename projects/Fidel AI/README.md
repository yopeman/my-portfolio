The **Fidel AI** project is a full-stack language learning platform that combines web, mobile, and backend services with AI-powered tutoring and community features.

### What it solves
It solves the need for a modern language learning ecosystem by integrating personalized lessons, AI chat, progress tracking, and course management into a unified platform.

### Technology Stack
* FastAPI, Python 3.12+
* MySQL, SQLAlchemy
* React 18, Vite, Tailwind CSS
* React Native, Expo, NativeWind
* LangChain, Ollama
