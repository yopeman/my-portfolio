The **AI-Powered Learning Platform (AiPLP)** project is a full-stack educational ecosystem for students, teaching assistants, and administrators.

### What it solves
AiPLP solves the problem of fragmented learning systems by centralizing AI-generated course content, role-based access, payment handling, certification, and interactive tutoring into one platform.

### Technology Stack
* Node.js, Express, Sequelize
* React
* React Native (Expo)
* MySQL
* OpenAI, Google GenAI
* Chapa
