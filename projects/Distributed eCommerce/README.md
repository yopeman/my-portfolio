The **Java-eCommerce-Console-App** project is a distributed client-server e-commerce system built with Java and Java RMI.

### What it solves
It solves the need for a remote shopping and order management system by demonstrating user authentication, a product catalog, shopping cart flow, checkout, and server-client communication in Java.

### Technology Stack
* Java
* Java RMI
* SQLite
* Swing / JOptionPane
