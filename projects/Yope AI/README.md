The **Yope AI** project is a web-based intelligent assistant that supports chat, scheduling, and information lookup.

### What it solves
It solves the need for a simple AI companion by providing authenticated conversations, message history, and Google Gemini-powered response generation.

### Technology Stack
* Python, Flask, Flask-CORS
* MySQL
* AngularJS, HTML, CSS, JavaScript
* Google Gemini API
