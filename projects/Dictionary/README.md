The **Dictionary-App-By-Flask-API-and-Angular** project is a bilingual English-Amharic dictionary app that pairs a Flask backend with an AngularJS frontend.

### What it solves
It solves the need for an end-to-end translation and dictionary management tool by providing word lookup, CRUD operations, and an administration dashboard.

### Technology Stack
* Python, Flask, Flask-CORS
* SQLite3
* AngularJS, HTML, JavaScript
