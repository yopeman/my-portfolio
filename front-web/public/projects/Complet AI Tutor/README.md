The **complete_ai_tutor** project is a multi-agent autonomous learning platform that uses specialized AI agents to plan lessons, tutor learners, evaluate progress, and generate certificates.

### What it solves
It solves the limitations of one-size-fits-all education by delivering personalized, mastery-based study paths with AI-generated exams, interactive content, and adaptive feedback.

### Technology Stack
* React, Tailwind CSS
* FastAPI, SQLAlchemy, MySQL
* LangChain, Groq
* Chapa, YouTube Search, Google Search, Translate
