The **smart-todo-app** project is an AI-powered project and task management platform with web and mobile interfaces.

### What it solves
It solves the need for intelligent project coordination by automating task creation, updates, reporting, and voice-enabled interaction in a collaborative workspace.

### Technology Stack
* Node.js, Express, Apollo Server, TypeScript
* React, Tailwind CSS
* React Native, Expo, NativeWind
* PostgreSQL, Sequelize
* Ollama, LangChain, LangGraph
