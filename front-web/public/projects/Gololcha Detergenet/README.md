The **Gololcha-Detergent-Backend** project is a backend API for an e-commerce platform focused on detergent products.

### What it solves
It solves the need for an online retail backend by handling user management, product inventory, orders, payments, and notifications for a detergent e-commerce store.

### Technology Stack
* Node.js, Express
* Sequelize
* JavaScript
