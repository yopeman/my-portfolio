The **JavaFX-Chat-App** (ChatFX) project is a desktop chat application built with JavaFX.

### What it solves
It solves the need for a real-time messaging client by providing a desktop chat interface, network communication, and user profile support in Java.

### Technology Stack
* Java
* JavaFX
* Java networking / sockets
