The **entrance_exam_app** project is an AI-powered platform for entrance exam preparation, built to analyze study materials and generate practice assessments.

### What it solves
It solves the challenge of exam readiness by automatically creating practice exams, grading answers, and providing interactive study support from uploaded educational content.

### Technology Stack
* Python
* FastAPI
* AI-powered content generation
* Document ingestion and evaluation services
